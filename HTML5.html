Règles de la base HTML 5

Un documents HTML doit respecter certines règles simples


  • Les éléments en paire doivent avoir une balise d'ouverture et une balise de fermeture.
    Cela facilite la tâche des navigateurs. Par exemple, plutôt que d'écrire :< li> Item 1
    On préférera le code suivant : < li> Iteml < /li>

  • Les éléments eurphelin ne comportent qu'une seule balise et il est plus correct de les terminer
    par les caractères précédés d'un espace pour marquer la fin de l'élément
    Par exemple, plutôt que d'écrire :
    on préférera le code suivant : < br/>

Balise Description
< table > Tableau
< tr > Ligne de tableau
< th > Cellule d'en-tête
< td > Cellule
